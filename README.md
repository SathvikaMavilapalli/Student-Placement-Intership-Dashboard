# 🎓 Student Placement & Internship Dashboard

## 📌 Overview
The **Student Placement & Internship Dashboard** is an interactive Power BI project that provides insights into student placements and internship progress across different departments. The dashboard helps colleges and placement cells track placements, internship status, offers received, and identify reasons why students remain unplaced.


## 🚀 Features
- 📊 Total Students KPI
- ✅ Students Placed KPI
- 🎓 Internship Completed Count
- ⏳ Internship Ongoing Count
- ❌ Internship Not Started Count
- 📈 Department-wise Student Distribution
- 💼 Offer Received Analysis
- 📊 Internship Status Analysis
- 📋 Student Details Table
- 📉 Reasons for Not Being Placed
- 🎛 Interactive Slicers for:
  - Department
  - Year/Batch
  - Internship Status
  - Placement Attended
  - Offer Received
  - Internship Interest
    

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel
- Power Query
- DAX
- Data Modeling


## 📊 Dashboard KPIs
- Total Students
- Students Placed
- Internship Completed
- Internship Ongoing
- Internship Not Started


## 📈 Insights
- Tracks placement performance across departments.
- Monitors internship completion and ongoing internships.
- Analyzes offer received percentage.
- Identifies common reasons for students not getting placed.
- Provides interactive filtering for better analysis.


## 🎯 Objective
The goal of this project is to create an interactive dashboard that helps educational institutions monitor placement activities, internship progress, and student performance using data visualization techniques.
